# Iris_EDA
# 🌸 Iris Dataset Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the famous Iris dataset using Python. The goal is to understand the structure, distribution, and relationships between features, as well as prepare the dataset for future machine learning tasks.

---

## 📊 Dataset Information
- Total Records: 150  
- Total Features: 6  
- Features:
  - SepalLengthCm
  - SepalWidthCm
  - PetalLengthCm
  - PetalWidthCm
  - Species (Target Variable)
  - Id  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- Matplotlib  

---

## 🔍 Analysis Performed
- Data loading and inspection  
- Checking dataset shape and columns  
- Missing values and duplicates detection  
- Statistical summary  
- Target variable distribution  
- Histogram visualization  
- Skewness analysis  
- Boxplots for feature comparison  
- Correlation matrix and heatmap  
- Outlier detection  

---

## 📈 Key Observations
1. The dataset is clean with no missing values or duplicates.  
2. The target classes are perfectly balanced.  
3. Petal measurements show more variability than sepal measurements.  
4. Iris-setosa is clearly separable, while the other two species overlap slightly.  
5. Petal length and width have strong positive correlation.
